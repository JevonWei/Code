2. 猜数字游戏  
    a. 生成一个[0,100)随机数  
    b. 让用户重复猜五次(从，命令行输入一个整数)  
        猜的太大 -> 提示太大  
        猜的太小 -> 太小  
        猜中了 ->  猜中了  

        5次都没猜中 -> 退出，并提示太笨  
    
    调战：  
        当五次都没有猜中，提示太笨了，重新开始猜数游戏(重新生成随机数，重新开始5次计数)
